# 软件卸载 AppUninstaller V0.0.1

### 简介

从注册表读取当前安装的软件，并选择卸载，比控制面板列出的软件要多

### 功能

- 列出注册表中有的所有安装的软件
- 在输入框进行软件筛选
- 直接回车卸载列表中的第一个软件
- 点击卸载相应的软件
- 不支持 macOS
